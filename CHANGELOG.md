# Changelog

All notable changes to Mori ActionPoint will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-01-15

### Added
- Initial release of Mori ActionPoint
- Google Gemini 2.5 Flash AI integration for multimodal analysis
- Text input processing with AI-powered categorization
- Image upload and analysis using Gemini Vision
- Voice recording with audio transcription and analysis
- Automatic Philippine government agency suggestion
- Professional email template generation
- Editable report review system
- Email notification system for agencies and users
- Multi-input support (text, image, voice)
- React + Tailwind CSS frontend
- Node.js + Express backend
- Comprehensive documentation and setup guides
- Testing guides and API examples
- Security features (Helmet, CORS, rate limiting)
- Input validation with Joi
- File upload handling with Multer
- Responsive design for mobile and desktop

### Features
- **AI-Powered Analysis**: Google Gemini 2.5 Flash for intelligent report processing
- **Multi-Input Support**: Accept reports via text, image, or voice
- **Smart Agency Matching**: Automatically suggests relevant government agencies
- **Professional Templates**: Generates formal email templates
- **Editable Reports**: Review and modify AI-generated content
- **Email Notifications**: Send reports to agencies and confirmations to users

### Technical
- Google Gemini 2.5 Flash API integration
- Multimodal input processing (text, image, audio)
- RESTful API with Express.js
- React 18 with Vite
- Tailwind CSS for styling
- Nodemailer for email delivery
- Rate limiting (100 req/15min)
- File size limits (10MB)
- Environment-based configuration

### Documentation
- Complete setup guide
- Comprehensive testing guide
- API examples and documentation
- Quick reference for developers
- Deployment checklist
- Contributing guidelines
- Issue and PR templates

### Security
- Helmet security headers
- CORS protection
- Input validation
- File type and size restrictions
- Environment variable protection
- Error handling without sensitive data exposure
